### UTF-8 万岁
本目录提供了 UTF-8 格式的各节目源，可用于应付对编码要求严格的播放器或供 Git 比较用。   
目前采用如下指令人工生成：
```bash
find . -type f -name '*.txt' -exec sh -c "iconv -f GBK -t UTF-8 {}  > ~/temp" \; -exec mv ~/temp utf8/{} \;
iconv -f GBK -t UTF-8 data.csv > utf8/data.csv
```
